# Protein Kinase Prediction using Machine Learning

This project predicts kinase proteins from human protein sequences using machine learning.

Dataset:
- UniProt Human Proteome
- GO Annotations

Features:
- Protein Length
- Amino Acid Composition

Model:
- Random Forest
- SMOTE for class balancing

Results:
- Accuracy: 95.5%
- Precision: 27.6%
- Recall: 42.7%
- F1 Score: 0.335

Tools:
- Python
- Pandas
- Scikit-learn
- Biopython
